# Portfolio Website

フロントエンドエンジニア向けのポートフォリオサイトです。

## 技術スタック

- **Next.js 14** - React フレームワーク
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング
- **Ant Design** - UI コンポーネント
- **Three.js / React Three Fiber** - 3D グラフィックス

## 機能

- **Home**: 3D背景アニメーション付きのホーム画面
- **Profile**: プロフィール情報
- **Skills**: スキルセットの表示
- **Links**: ソーシャルリンク
- **Contact**: お問い合わせフォーム

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド

```bash
npm run build
npm start
```
